# nabd

Daily wird (ورد) companion for Muslims — helps you commit to a daily devotional routine,
track it, hold yourself accountable (محاسبة), and reflect on your worship over time.

نبض — "pulse". The daily wird is the believer's pulse; nabd helps you keep it steady.

## Status

Pre-bootstrap — generated by mimar. First milestones: ticket NBD-1 (bootstrap) and NBD-2
(structure + boundaries) in [`docs/backlog.md`](./docs/backlog.md).

## Stack

Next.js (App Router) · TypeScript strict · Tailwind + shadcn/ui · Dexie (local-first) +
Supabase (sync + Auth) · Zustand · Serwist (PWA) · Vitest + Playwright · pnpm · Vercel ·
Sentry + Vercel Analytics. Arabic-only, RTL-first.
